# Country Segmentation using K-Means Clustering

## Objective

Cluster countries based on socio-economic and health indicators and identify countries that should receive aid from HELP International.

## Techniques Used

* Data Preprocessing
* Feature Scaling
* K-Means Clustering
* PCA Visualization

## Results

Countries were segmented into:

1. Developed Countries
2. Developing Countries
3. Underdeveloped Countries

Cluster 1 was identified as the most vulnerable cluster due to low income, low GDP, low life expectancy, and high child mortality rates.

## Recommendation

HELP International should prioritize countries in Cluster 1 for aid allocation.
