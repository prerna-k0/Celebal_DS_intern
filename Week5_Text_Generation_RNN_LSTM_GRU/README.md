# 📘 Week 5 – Deep Learning Text Generation using RNN, LSTM and GRU

## Overview

This project implements and compares three recurrent neural network architectures—Vanilla RNN, LSTM, and GRU—for text generation. The models are trained on a text corpus to learn grammar, contextual dependencies, and sentence structure for next-word prediction.

## Objectives

* Perform text preprocessing using tokenization.
* Generate n-gram sequences for next-word prediction.
* Train Vanilla RNN, LSTM, and GRU models.
* Compare training loss and generated text quality.
* Understand long-term dependency learning in recurrent neural networks.

## Technologies Used

* Python
* TensorFlow / Keras
* NumPy
* Matplotlib

## Files

* `Week5_Text_Generation_RNN_LSTM_GRU.ipynb`
* `README.md`

## Results

* Vanilla RNN learned basic sequence patterns but struggled with long-term memory.
* LSTM generated more coherent and context-aware text using its gating mechanism.
* GRU achieved similar performance to LSTM with fewer parameters and faster training.
