# Week 8 - Single Agent System & Agent Pipeline

## Overview

This project demonstrates a simple rule-based single-agent pipeline.

The agent analyzes a user's query, determines its intent, routes it to the appropriate tool, and returns the result.

## Features

- Calculator Tool
- Keyword Extraction Tool
- General Response Tool
- Conditional Routing
- Rule-Based Agent Workflow

## Workflow

```
User Query
      ↓
Agent Router
      ↓
Select Appropriate Tool
      ↓
Generate Response
```

## Technologies

- Python
- Google Colab

## Tools Implemented

- Calculator
- Keyword Extractor
- General Response

## Conclusion

This project demonstrates the basic architecture of a single-agent pipeline using conditional routing and multiple tools.
