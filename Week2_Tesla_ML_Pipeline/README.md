# Tesla End-to-End ML Pipeline

## Objective

Build a complete Machine Learning pipeline on Tesla sales and production data.

## Dataset

Tesla EA Deliveries and Production Data (2015-2025)

## Steps Performed

1. Data Loading
2. Exploratory Data Analysis
3. Feature Engineering
4. Time Series Processing
5. Chronological Train-Test Split
6. Regression Modeling
7. Hyperparameter Tuning
8. Forecasting
9. Model Evaluation

## Algorithms Used

* Random Forest Regressor
* GridSearchCV

## Evaluation Metrics

* MAE
* RMSE
* R² Score

## Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
